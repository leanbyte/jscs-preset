# jscs-preset-leanbyte

Common JSCS configuration for LeanByte

## Usage

1. Add this module as a `devDependency`:

	`npm install -D jscs-preset-leanbyte`

2. Declare it as a preset in your `.jscsrc` file:

	```
	{
		"preset": "leanbyte",
		"excludeFiles": [
			<…>
		],
		"rules": [
			<…>
		]
	}
	```
